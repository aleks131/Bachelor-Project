Getting Started

Welcome to Smart Solutions by TripleA. This unified platform integrates Daily Planning, Image Galleries, and Performance Dashboards into one seamless experience.

Login

Access the system using your assigned username and password. Your role determines which applications you can access.

Dashboard

The central hub showing system status, notifications, and quick access to all your enabled applications.

Local Content

The system now uses optimized local folders for faster media loading and better reliability.

Role-Based Access

The system is secured with role-based permissions. Here is what each user type can see and do:

ADMIN Administrator

  • Full system access
  • User management (Create/Edit/Delete users)
  • System configuration & Monitoring
  • Access to all applications

MANAGER Manager

  • Performance Dashboard access
  • Gallery management
  • Can view but not edit system settings

OPERATOR Operator

  • Daily Plan Viewer access
  • View-only access to assigned content
  • Simplified interface for daily tasks

Core Functions

Daily Plan

Automated schedule display that changes based on time of day (Morning, Evening, Night shifts).

Gallery

High-performance image and video viewer with folder navigation and slideshow capabilities.

Performance

Visual KPI tracking with customizable widgets and real-time data updates.
