# Mockup Images

This directory contains visual mockups showing how each application mode works.

## Available Mockups

1. **login-screen.png** - Login interface with TripleA branding
2. **dashboard.png** - Main dashboard with application cards
3. **daily-plan-viewer.png** - Daily Plan Viewer showing schedule
4. **image-gallery.png** - Image Gallery with folder browsing
5. **performance-dashboard.png** - Performance Dashboard with KPI cards
6. **admin-panel.png** - Admin Panel for user management

## Usage

These mockups can be used for:
- Documentation
- Presentations
- User guides
- Marketing materials

All mockups are 1920x1080 resolution and show the dark theme interface.
