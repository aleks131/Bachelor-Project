## Step-by-Step Instructions

### Step 1: Go to Repository Settings
**Click this link**: https://github.com/aleks131/Bachelor-Project/settings

Or manually:
1. Go to: https://github.com/aleks131/Bachelor-Project
2. Click the **"Settings"** tab (top menu, next to "Code", "Issues", etc.)

### Step 2: Find Danger Zone
1. Scroll **ALL THE WAY DOWN** to the bottom of the settings page
2. Look for a red section labeled **"Danger Zone"**
3. It's the last section on the page

### Step 3: Change Visibility
1. In the Danger Zone, find **"Change visibility"**
2. Click the **"Change visibility"** button
3. A popup/modal will appear

### Step 4: Make Public
1. In the modal, you'll see options:
   - "Make private"
   - **"Make public"** ‚Üê Select this one
2. Read the warning message
3. Type the repository name: `aleks131/Bachelor-Project`
4. Click **"I understand, change repository visibility"**

### Step 5: Verify
1. Go back to repository main page: https://github.com/aleks131/Bachelor-Project
2. Look at the top right, next to the repository name
3. You should see a **"Public"** badge (green/blue)

## Troubleshooting

### Can't See "Change Visibility" Button?
- **Check**: Are you the repository owner?
- **Check**: Are you logged into GitHub?
- **Check**: Did you scroll all the way down?

### Repository Name Confirmation Fails?
- Try: `aleks131/Bachelor-Project` (without dash)
- Try: `Bachelor-Project` (just the name)
- Check for typos

### Still Shows Private After Change?
- Wait 1-2 minutes (GitHub needs time to update)
- Refresh the page
- Clear browser cache
- Try incognito mode
